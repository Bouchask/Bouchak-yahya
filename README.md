## 🛠 Tech Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Radix UI (Shadcn/UI)
- **State Management**: React Context API (for Language/Theme)
- **Build Tool**: Vite

## ✨ Key Features
- **Dual Language Support**: Seamlessly switch between English and French.
- **Dark/Light Mode**: Fully responsive theme toggle with persistent storage.
- **Glassmorphism Design**: Modern, clean UI using translucent elements and blurred backgrounds.
- **Responsive Layout**: Optimized for all devices (Mobile, Tablet, Desktop).
- **CV Download**: Integrated download for both English and French versions of my CV.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or bun

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Bouchask/Bouchak-yahya.git
   ```
2. Install dependencies:
   ```bash
   npm install
   # or
   bun install
   ```

### Running the Development Server
```bash
npm run dev
# or
bun dev
```
The site will be available at `http://localhost:8080`.

### Building for Production
```bash
npm run build
```
